## 💻 Sobre o Projeto

O Daily Diet é uma aplicação mobile construída em React Native com Typescript para gerenciar as refeições que são feitas diariamente pelo usuário, e com base nas informações disponibilizadas pelo o usuário se é uma refeição que está dentro da refeição ou não, o app calcula em forma de porcentagem a media geral de quantas refeições estão dentro da dieta. A aplicação usa vários conceitos vistos em aula como Estados, Imutabilidade do estado, Listas e chaves no React Native, Propriedades, Componentização e Parâmetros de navegação. O projeto compõe um dos desafios da trilha de React Native do Ignite da Rocketseat.

## 🔨 Funcionalidades do projeto

- [ ] Adicionar uma nova refeição
- [ ] Editar uma refeição
- [ ] Deletar uma refeição na listagem
- [ ] Mostrar as estatísticas do progresso da dieta
- [ ] Navegação entre telas em pilha
- [ ] Armazenamento local das refeições com Async Storage

## ✔️ Tecnologias utilizadas

- `React Native`
- `Typescript`
- `NodeJS 18.16.0`

## 🛠️ Instalação e uso

Você pode baixar o projeto e rodar em seu computador seguindo os passos abaixo:

[IgniteGym](https://github.com/bandeirapk/daily-diet/archive/refs/heads/main.zip) - Baixar o projeto

```bash
# Clone este repositório
git clone https://github.com/bandeirapk/daily-diet.git

# Acesse a pasta mobile do projeto
cd daily-diet/

# Instale as dependências
npm install

# Rode o projeto
npx expo start

# Em seguida, abra o emulador ou conecte seu dispositivo e rode o projeto.
```
